# v0.7.3

This release requires a simple async update.

Just stop your node and replace the binary with [wardend v0.7.3](https://github.com/warden-protocol/wardenprotocol/releases/tag/v0.7.3).

:::warning
If you used **v0.7.2** to generate keys (addresses), generate new keys using **v0.7.3** or newer and move any existing funds there.

**Keys generated with v0.7.2 are not valid anymore.**
:::
